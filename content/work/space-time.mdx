## Role / Scope

Personal experimental project — infinite-scroll website with custom drag/scroll navigation built around a 10+ year personal archive of architecture images. Pulls together an architecture degree background with interaction and web design. Built as a Claude Code experiment.

## Goals

- Design an unconventional browsing experience for a large image archive — drag-and-scroll exploration over traditional grids or feeds.
- Use the build to test Claude Code as a development partner on something with novel interaction patterns.
- Create a personal space to display and engage with a long-running image collection.

## Outcomes

- Working infinite-scroll site with custom navigation.
- Personal archive surfaced in a browseable, exploratory format.
- Hands-on Claude Code experience on a non-trivial interaction.
